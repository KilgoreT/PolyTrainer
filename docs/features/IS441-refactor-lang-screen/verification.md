# IS441. Верификация — чеклист

## Онбординг (первый запуск)

- [ ] Splash → экран создания словаря (форма, без AppBar)
- [ ] Создать словарь без языка → попадаем на главный экран
- [ ] Создать словарь с языком и флагом → попадаем на главный экран
- [ ] Созданный словарь отображается в dropdown AppBar
- [ ] Системная кнопка "назад" на форме онбординга — выход из приложения (не краш)

## Создание словаря из dropdown

- [ ] Dropdown в AppBar (словарь/квиз/статистика) → "Добавить словарь" → форма с AppBar
- [ ] Создать словарь → возврат на предыдущий экран (popBackStack)
- [ ] Кнопка "назад" в AppBar → возврат без создания
- [ ] Созданный словарь появляется в dropdown

## Создание словаря из списка

- [ ] Настройки → "Все словари" → экран списка с AppBar
- [ ] Кнопка "Новый словарь" → форма создания с AppBar
- [ ] Создать словарь → возврат в список
- [ ] Список обновлён (новый словарь виден)
- [ ] Кнопка "назад" в форме → возврат в список без создания

## Список словарей

- [ ] Отображает все словари (название, флаг если есть)
- [ ] Кнопка "назад" в AppBar → возврат в настройки
- [ ] Пустое состояние если нет словарей ("Нет словарей / Создайте первый!")

## Выход из приложения при пустом списке

- [ ] Список словарей → удалить все словари → нажать "назад" → приложение закрывается
- [ ] При следующем запуске → splash → onboarding (создание словаря)

## Обновление списка при возврате

- [ ] Список → "Новый словарь" → создать → вернуться → новый словарь виден в списке
- ✅ Список обновляется автоматически (через Flow, без ручного reload) — реализовано через DictionaryListFlowHandler

## Удаление словаря

- [ ] Иконка удаления → диалог подтверждения
- [ ] "Удалить" → словарь удалён, список обновлён
- [ ] "Отмена" → диалог закрыт, ничего не удалено
- [ ] Удаление текущего словаря → переключение на первый оставшийся
- [ ] Удаление единственного словаря → пустое состояние

## Форма создания/редактирования

- ✅ Поле названия — ввод текста, кнопка "Создать" активна при непустом названии — reducer updateName + saveButtonEnabled
- ✅ Пустое название / только пробелы → кнопка неактивна — isNotBlank() в extension
- ✅ Чекбокс "Привязать к языку" — по умолчанию выключен — isLanguageBound = false default
- ✅ Включение чекбокса → появляется поле выбора языка — toggleLanguageBound extension
- ✅ Выключение чекбокса → язык и флаг сбрасываются — toggleLanguageBound clears selectedLanguage/Flag

## Выбор языка

- [ ] Тап на поле языка → bottom sheet со списком языков
- ✅ Фон bottom sheet — белый, текст чёрный — containerColor = whiteColor, color = secondary
- [ ] Поле поиска с автофокусом, клавиатура появляется
- ✅ Фильтрация по названию на языке устройства ("исп" → "Испанский") — filterLanguages extension, contains ignoreCase
- ✅ Выбор языка → bottom sheet закрывается, название в поле — selectLanguage extension, show = false

## Выбор флага

- [ ] После выбора языка → горизонтальный ряд флагов с названиями стран
- ✅ Названия стран на языке устройства — getLocalizedCountryName через Locale
- [ ] Горизонтальный скролл если много стран
- ✅ Тап на флаг → выделение рамкой — BorderStroke в FlagGridWidget
- ✅ Один флаг (например японский) → автовыбор — updateFlags: if (list.size == 1) list.first()

## Переключение словарей

- [ ] Dropdown в AppBar → список словарей → тап → переключение
- [ ] После переключения: словарь, квиз, статистика загружают данные нового словаря
