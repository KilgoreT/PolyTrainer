# IS441. Задача 3 — Полноценное управление словарями

## Описание

Экран Dictionary Management — полноценное управление словарями: просмотр, создание, редактирование, удаление.

## Требования

### 1. Список существующих словарей

- Показывать все словари пользователя
- Для каждого словаря: флаг (если есть), название, язык (если привязан)
- Если словарей нет — пустое состояние с предложением создать первый

### 2. Удаление словаря

- Иконка удаления на элементе списка (без свайпа)
- **Обязательное подтверждение** — диалог с предупреждением: удаление словаря удалит все слова, лексемы и квизы в нём
- После удаления: если удалён текущий словарь → переключиться на первый оставшийся
- Если словарей не осталось → показать пустое состояние

### 3. Редактирование словаря

- Тап на словарь → режим редактирования
- Можно изменить: название, привязку к языку, флаг
- Сохранение изменений

### 4. Создание нового словаря

#### 4.1 Название

- Текстовое поле, обязательно
- Пользователь вводит что угодно: "English", "Биология", "Сленг" и т.д.

#### 4.2 Привязка к языку (опционально)

- Чекбокс/переключатель "Привязать к языку"
- По тапу — появляется выпадающий список языков
- Список языков: все языки мира из `Locale.getISOLanguages()`
- Отображение на языке устройства (`locale.displayLanguage`)
- Поле поиска/фильтрации по названию (на любом языке)
- Выбор языка → сохраняется привязка

#### 4.3 Выбор флага (опционально, после выбора языка)

- Появляется после выбора языка
- Показывает флаги стран, где говорят на выбранном языке
- Если страна одна (например, японский → Япония) — автовыбор
- Если стран несколько (испанский → Испания, Мексика, Аргентина...) — список для выбора
- Если язык не выбран — флаг недоступен

#### 4.4 Кнопка "Создать"

- Активна когда заполнено название
- Создаёт словарь → добавляет в список → делает текущим
