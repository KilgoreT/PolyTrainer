## Задача

Захожу в список словарей. Там словарь без флага — соответственно буква на сером фоне. Захожу в настройки словаря, устанавливаю флаг. Возвращаюсь в список словарей. В списке словарей флаг виден. Переключаюсь на вкладку где в аппбаре можно выбирать словари, а там все еще старая иконка словаря.

## Контекст

- Тикет: IS445
- Ветка: IS445-stale-dictionary-icon
- Баг: иконка словаря в аппбаре другой вкладки не обновляется после смены флага в настройках словаря. Список словарей обновляется корректно, но picker словарей в аппбаре показывает устаревшую иконку (букву на сером фоне вместо флага).
