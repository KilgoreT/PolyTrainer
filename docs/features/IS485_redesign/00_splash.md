# 00 — Splash

Скриншот: [00_splash.png](00_splash.png).

## Назначение

Первый экран при запуске приложения. Показывается на время инициализации (DI, БД), затем автоматически сменяется главным экраном («Словарь») либо экраном создания словаря, если словарей ещё нет. Никакого взаимодействия — чисто брендовый экран.

## Структура

Полноэкранная одноцветная заливка, по центру — логотип.

## Элементы

### Логотип (капибара)

- **Смысл:** бренд приложения Lexeme — белый линейный (outline) силуэт головы капибары с одной цветной точкой (нос/глаз-акцент).
- **Текст:** отсутствует. На текущем экране нет ни названия приложения, ни слогана.
- **Доменная сущность:** — (бренд).
- **Значимость:** primary (единственный элемент экрана).
- **Поведение:** статичный; анимации в текущей версии нет.

### Фон

- **Смысл:** брендовый цвет (индиго/фиолетовый, тот же семантический primary-цвет, что у FAB и кнопок на остальных экранах).
- **Значимость:** primary — задаёт бренд-цвет всей палитры приложения.

## Заметки для дизайна

- Splash — точка, где фиксируется палитра: primary-цвет фона здесь = цвет акцентных элементов (FAB, активные кнопки, чипы) на всех остальных экранах. Редизайн палитры должен начинаться отсюда.
- Название приложения (Lexeme) на splash сейчас не показывается — решить, нужно ли добавлять.
- Учитывать системный Android splash (API 31+): иконка в круге на фоне — логотип должен читаться и в этом формате.
