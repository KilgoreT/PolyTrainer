## Ручное тестирование

### Онбординг: новый вид первого экрана
1. Удалить все словари (или чистая установка приложения), перезапустить приложение.
2. Дождаться перехода со splash на экран создания словаря.
3. Ожидание: тёплый почти-белый фон; вверху контента заголовок «Новый словарь» и подзаголовок «Выберите язык и назовите словарь»; AppBar и кнопка «назад» отсутствуют; внизу кнопка «Создать» disabled (приглушённая, без тени), с отступами от краёв.
4. Логи:
   - `###DictForm### экран формы словаря открыт в режиме онбординга`

### Create-режим: заголовок в AppBar
1. Имея хотя бы один словарь, открыть создание нового: дропдаун словарей → «Добавить словарь» (или «Все словари» → добавить).
2. Осмотреть верх экрана.
3. Ожидание: AppBar с заголовком экрана и кнопкой «назад»; внутреннего дубля заголовка в контенте НЕТ; подзаголовок «Выберите язык и назовите словарь» — в контенте; остальной вид как в онбординге; «назад» возвращает на предыдущий экран.
4. Логи:
   - `###DictForm### экран формы словаря открыт в режиме создания`

### Edit-режим: префилл и «Сохранить»
1. Открыть «Все словари», тапнуть по существующему словарю.
2. Осмотреть экран.
3. Ожидание: AppBar с заголовком редактирования (не «Новый словарь»); поле имени префиллено, флаг словаря отмечен в сетке и показан в превью с двойным кольцом; кнопка внизу — «Сохранить», enabled.
4. Логи:
   - `###DictForm### экран формы словаря открыт в режиме редактирования, словарь загружен`

### Ввод имени: Material-поле и live-превью
1. Открыть экран создания словаря (любой режим, флаг не выбран).
2. Тапнуть в поле названия, ввести «Английский» посимвольно.
3. Ожидание: над полем uppercase-лейбл «НАЗВАНИЕ СЛОВАРЯ» акцентным цветом, underline и курсор акцентные; превью слева показывает первую букву («А») сразу с первого символа и обновляется live; кнопка «Создать» становится enabled (цветная, с тенью) от первого символа; при полном стирании имени превью — пустой круг, кнопка снова disabled.
4. Логи:
   - `###DictForm### название словаря изменено`

### Выбор флага: маркер в сетке и превью
1. Открыть экран создания словаря.
2. Осмотреть сетку, тапнуть по флагу (например, Великобритания), затем тапнуть по нему повторно.
3. Ожидание: сетка — 4 колонки, круглые флаги 56dp, подписи под флагами переносятся на новую строку без «…»; после тапа у выбранного флага кольцо акцентного цвета с внутренним зазором, бейдж-галочка в правом нижнем углу, подпись акцентная bold; превью слева — этот флаг с двойным кольцом; повторный тап снимает выбор (маркер исчезает, превью — буква/пустой круг).
4. Логи:
   - `###DictForm### флаг выбран/выбор снят`

### Поиск по флагам: фильтрация
1. Открыть экран создания словаря.
2. Осмотреть поле поиска, ввести «Фран» (на EN-локали проверить hint отдельно).
3. Ожидание: поиск — пилюля на светло-серой подложке с иконкой-лупой и hint «Поиск по стране или языку…» (EN: hint по смыслу совпадает с RU, не «Filter flags…»); сетка фильтруется по мере ввода (остаётся Франция и совпадения); поле можно очистить, сетка возвращается к полной.
4. Логи:
   - `###DictForm### фильтр флагов изменён`

### Поиск: «Ничего не найдено»
1. Открыть экран создания словаря.
2. Ввести в поиск бессмысленный запрос, например «яяяя».
3. Ожидание: сетка пуста, по центру области сетки вторичным текстом «Ничего не найдено» (на EN-локали — английский вариант); никаких пустых мерцаний при старте экрана быть не должно (текст показывается только при непустом запросе); очистка запроса возвращает сетку.
4. Логи:
   - `###DictForm### поиск флагов без результатов`

### Кнопка «Создать/Сохранить»: вид и результат
1. Открыть экран создания словаря, убедиться что имя пустое — осмотреть кнопку.
2. Ввести имя, выбрать флаг — осмотреть кнопку снова.
3. Нажать кнопку.
4. Ожидание: при пустом имени кнопка приглушённая (disabled, без цветной тени), форма — radius 16, высота 56, отступы от краёв (не прижата вплотную); при непустом имени — акцентный цвет и цветная тень под кнопкой; после нажатия словарь создан (онбординг → главный экран; create → возврат назад, новый словарь текущий; edit → изменения сохранены).
5. Логи:
   - `###DictForm### сохранение словаря запрошено`
