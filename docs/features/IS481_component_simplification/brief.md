**Ветка:** `IS481_lexeme_component_constructor` (коммитим сюда же)

Архитектуру **не трогаем** (глобальные компоненты, Scope, привязка к словарям остаются как есть). Фича = переименование терминов + реструктуризация доков + синхронизация спек с кодом.

## Терминология (принято)

- **Охват** (`scope`) — к каким словарям относится компонент.
  - **Общий** (`global`) — во все словари.
  - **Пословарный** (`per-dictionary`) — привязан к выбранным словарям.
- **Несколько значений** (`Multiple values`; код: `isMulti` → `isMultiple`) — может ли лексема иметь несколько значений этого компонента (vs одно).
