# Task

## IS453. WordCard: expand input field width, update confirm icon

UX-фикс экрана WordCard — поля ввода перевода и определения.

## Проблема

1. **Поле ввода слишком узкое** — занимает минимальную ширину по контенту. Трудно попасть для фокуса.
2. **Иконка подтверждения невнятная** — текущая иконка не считывается как "применить". Нужна зелёная круглая с галочкой.
3. **Иконка двигается** — находится рядом с полем ввода и смещается по мере набора текста.

## Требования

- Поле ввода перевода/определения — максимальная ширина (fillMaxWidth или weight(1f))
- Иконка подтверждения: зелёный круг с белой галочкой, фиксированная позиция (не двигается при вводе)
- Затрагивает: экран WordCard, виджеты добавления/редактирования перевода и определения
