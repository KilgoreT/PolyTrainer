# Задача: IS441 — dictionary-flag-picker

## Задача

На экране добавления словаря убрать привязку к языку. Оставить выбор флагов. Должно быть поле фильтрации флагов по стране и по языку. Поле одно — фильтрует по обоим критериям. Если флаг выбран — показан рядом с полем ввода названия. Если не выбран — placeholder.

## Критерии приёмки

1. Привязка к языку убрана с экрана создания словаря.
2. Выбор флагов работает.
3. Одно поле фильтрации фильтрует флаги по названию страны и по языку.
4. Выбранный флаг отображается рядом с полем ввода названия.
5. Если флаг не выбран — показан placeholder.
