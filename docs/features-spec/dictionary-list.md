# Список словарей — Спецификация

Экран просмотра, удаления словарей и навигации к созданию/редактированию.

---

## Навигация (routes)

Три точки входа — три route, один экран, разное поведение:

- **DICTIONARY_SETUP** — онбординг (первый запуск). Нет AppBar. После "Далее" → главный экран.
- **DICTIONARY_LIST** — из настроек. AppBar с "назад". При пустом списке "назад" → выход из приложения.
- **DICTIONARY_CREATE** — из dropdown AppBar (на вкладках). Открывает сразу форму, не список.

Route определяет callbacks: `onClose`, `onBackPress`, `onExit`.

---

## Список

- Отображает все словари пользователя
- Каждый элемент: флаг (или placeholder) + название + иконка удаления
- Тап на элемент → форма редактирования (навигация к DictionaryForm)
- Тап на иконку удаления → диалог подтверждения
- Кнопка "Новый словарь" внизу → форма создания

### Пустое состояние

Если словарей нет — "Нет словарей / Создайте первый!" по центру + кнопка "Новый словарь".

### Кнопка "Далее" (только SETUP route)

- Появляется после создания хотя бы одного словаря
- `setCurrentDictionary(id)` → `onClose()` → главный экран

---

## Удаление

1. Тап на иконку удаления → диалог `AlarmDialogWidget`
2. "Удалить" → CASCADE delete (словарь → words → lexemes → write_quiz)
3. Если удалённый был текущим → переключиться на первый оставшийся
4. Если словарей не осталось → пустое состояние

---

## Обновление списка

Реактивное через `flowDictionaryList()` (Room Flow → MateFlowHandler → автоматическое обновление UI).

---

## Навигация "назад" при пустом списке

- DICTIONARY_LIST + пустой список + "назад" → `onExit` → `Activity.finish()`
- При следующем запуске → splash → нет словарей → DICTIONARY_SETUP

---

## Порядок словарей

По дате добавления (Room default ordering).

---

## UI

### AppBar

- SETUP: без AppBar (`onBackPress = null`)
- LIST: AppBar с "← Словари"
- Наличие определяется nullable `onBackPress`

### Элемент списка

`Row`: ImageFlagWidget (или placeholder) + название (secondary color) + IconBoxed удаления

### Диалог удаления

`AlarmDialogWidget`: "Удалить словарь? Все слова, определения и результаты квизов будут удалены."
